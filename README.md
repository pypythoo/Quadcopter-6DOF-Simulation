# Quadcopter 6-DOF Simulation using MATLAB

## Overview
This project simulates a quadcopter using a six-degree-of-freedom nonlinear dynamic model. A PID controller is used to regulate altitude and attitude, while a motor mixing algorithm converts thrust and torque commands into motor speeds.

## Features
- 6-DOF nonlinear quadcopter dynamics
- PID altitude and attitude controller
- Motor mixing algorithm
- MATLAB ODE45 numerical integration
- Flight response plots

## Files

- main.m
- params.m
- pid_controller.m
- motor_mixing.m
- quad_dynamics.m
- plot_results.m

## How to Run
The simulation will generate flight plots and save the results in `sim_results.mat`.
